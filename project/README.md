# Proyecto Final – Sistemas Inteligentes

## Descripción General

Este proyecto se centra en la aplicación de técnicas de aprendizaje profundo (Deep Learning) para la identificación automática de malware a partir de archivos o patrones de comportamiento.

A diferencia de los sistemas tradicionales basados en firmas que solo detectan amenazas conocidas, el enfoque propuesto busca detectar variantes desconocidas y ataques de día cero, gracias a la capacidad de los modelos de aprendizaje profundo para extraer y generalizar patrones complejos presentes en los datos.

## Objetivo del Proyecto

Diseñar e implementar un modelo de aprendizaje profundo que permita:

* Clasificar archivos o comportamientos de red como maliciosos o benignos.
* Detectar variantes nuevas de malware que no aparecen en bases de datos de firmas.
* Contribuir a la automatización de procesos de ciberseguridad mediante inteligencia artificial.

## Tecnologías y Herramientas

* Lenguaje: Python
* Frameworks: TensorFlow / Keras / PyTorch
* Bibliotecas auxiliares: NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn
* Dataset: Repositorios públicos de malware (p. ej. EMBER, VirusShare, Malimg Dataset)
